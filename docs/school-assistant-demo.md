School Assistant
Vipin Vijayan Nair

The Challenge
Challenge
Parents receive numerous emails from schools about different events and programs
Gap
Information is scattered across emails, making it hard to remember and find specific details
Barrier
No centralized way to search through all school-related communications
Need
Simple way to ask questions and get answers about school events
Impact
Parents miss important events/programs/assignments or spend too much time searching

Our Answer
Solution Overview
Approach
AI-powered chat application where parents can ask questions naturally
Method
Smart system that searches multiple sources: local files, Gmail, and web
Focus
Ask anything like "What karate classes are available?" and get instant answers
Delivery
User-friendly chat interface with real-time responses
Outcome
Find school related information in seconds, not minutes or hours

Technology Stack
AI Platform
OpenAI GPT-4 for intelligent responses
Search System
Multi-agent setup (Local Database + Gmail + Web Search)
Backend
Python with FastAPI, Quadrant, Langraph, RAGAS etc
Frontend
React web application
Data Storage
Vector database for fast semantic search
Evaluation
RAGAS framework to measure quality
