School Events RAG — Multi-Agent Capstone

Production-style RAG application that helps parents find school events and programs via natural-language questions. A multi-agent system orchestrated by LangGraph routes queries between a local Qdrant vector store of school data and the live web (Tavily) with fallback logic.

  • RecursiveCharacterTextSplitter (1000/200 chunks) — improved recall by 2× over full-doc indexing
  • Multi-agent orchestration: LocalEvents agent + WebSearch agent with LangGraph routing
  • Gmail MCP integration — searches parent's Gmail for school emails via Model Context Protocol
  • FastAPI backend + React frontend, deployable to Railway & Vercel
  • Designed for production: env-driven config, OAuth, deployment scripts, Docker support
